# THERABOT — Healthcare Chatbot

A healthcare-focused chatbot that answers health-related queries across three categories: symptoms, medications, and general wellness. Think of it as a medically-focused conversational assistant.

---

## Features

- Real-time message handling with sub-second response times
- Responsive chat interface with typing indicators and message history
- Handles symptoms, medications, and general wellness questions
- Input validation tested across 3 browsers
- Processed 100+ test conversations

---

## Tech Stack

- **Backend:** Node.js, Express
- **AI:** OpenAI API
- **Frontend:** HTML, CSS, JavaScript

---

## Getting Started

1. Clone the repo
   git clone https://github.com/Jenny2502/THERABOT.git

2. Install dependencies
   npm install

3. Run the app
   node server.js

4. Open your browser and go to
   http://localhost:3000
