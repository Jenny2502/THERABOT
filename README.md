# THERABOT — Healthcare Chatbot

A healthcare-focused chatbot built with Node.js and the OpenAI API, designed to answer health-related queries across three categories: symptoms, medications, and general wellness.

---

## Features

- Real-time message handling with sub-second response times
- Conversational AI powered by OpenAI API
- Responsive chat interface with typing indicators and message history
- Input validation tested across 3 browsers
- Handles 100+ test conversations across 3 health query categories

---

## Tech Stack

- **Backend:** Node.js, Express
- **API:** OpenAI API
- **Frontend:** HTML, CSS, JavaScript

---

## Getting Started

### Prerequisites

- Node.js installed
- OpenAI API key

### Installation

1. Clone the repo
   git clone https://github.com/Jenny2502/THERABOT.git

2. Install dependencies
   npm install

3. Add your OpenAI API key — create a .env file in the root folder
   OPENAI_API_KEY=your_api_key_here

4. Run the app
   node server.js

5. Open your browser and go to
   http://localhost:3000
